Top 20 programming languages right now

The TIOBE Index is a widely recognized ranking of programming languages based on their popularity and usage. It is updated monthly and provides insights into the trends and popularity of different programming languages in the software development industry.

The TIOBE Index uses various factors to determine the rankings, including search engine queries, job postings, and the number of skilled engineers, courses, and third-party vendors related to a particular programming language. The index assigns a score to each language, indicating its popularity and market demand.

Fun fact about Python

Python is named after a TV show

Python seems like an odd name for a programming language. There is no truth to the popular belief that Python is named after the famous non-venomous snake of the same name. It has a whole different story to tell. Guido was a huge fan of “Monty Python’s Flying Circus”, a comedy TV show on the BBC. Back in the 70s, Monty Python was a renowned British comedy group that performed in British colonies. In honor of Monty Python’s Flying Circus, Guido decided to name the language Python, since he was a big fan.
